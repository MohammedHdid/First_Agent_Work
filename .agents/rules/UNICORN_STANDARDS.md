# THE UNICORN STANDARDS

These are the non-negotiable quality levels for any output generated by the Autonomous Landing Page Architect.

## 1. Performance
- **Lighthouse Score**: Must be > 95 across all metrics (Performance, Accessibility, Best Practices, SEO).
- **Asset Optimization**: All images must be WebP/AVIF. Lazy loading is mandatory. Server-side rendering (SSR) or Static Site Generation (SSG) via Next.js where applicable.

## 2. Accessibility
- **Standard**: WCAG 2.1 AA Compliance.
- **Requirements**: Proper ARIA labels, semantic HTML tags, keyboard navigability, and high contrast ratios.

## 3. Design Aesthetics
- **Core Theme**: "Unicorn" (Ferrari performance + Tractor utility).
- **Visuals**: Use Glassmorphism (translucency and background blur).
- **Lighting**: Implement Light-leaks and subtle gradient meshes.
- **Motion**: Framer Motion transitions for all page entrances and micro-interactions. Elements should feel responsive and "alive".
- **Typography**: Modern, premium sans-serif (e.g., Inter, Outfit) with perfect typographic hierarchy.
