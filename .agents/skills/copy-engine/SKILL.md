# Copy Engine Skill: Context-Aware Conversion

**Logic:** Produce highly persuasive, context-aware copywriting based on the `STRATEGY_MAP.md`. Focus the tone based on the niche:
- **SaaS:** Benefit-driven, metric-focused (e.g., "Save 10 hours a week").
- **E-Com:** Offer-driven, urgency-focused (e.g., "Limited drop, secure yours now").
- **Coaching/Influencer:** Transformation-driven, empathetic (e.g., "Finally break through your revenue ceiling").

**Instruction:** 
You must use the "Problem-Agitate-Solve" (PAS) framework for EVERY Hero section.
1. **Problem:** Call out the pain point explicitly.
2. **Agitate:** Twisting the knife on why this problem costs them money or time.
3. **Solve:** Introduce the product/service as the inevitable Unicorn solution.

**Error Handling:**
If the provided niche context is too sparse to write compelling copy, infer standard pain points for that industry and explicitly note these assumptions as comments in the code or a review document so the user can refine them later.
