# Evolution Skill: Self-Improvement

**Logic:** You are commanded to research new 2026 design trends and technical frameworks on a weekly basis.

**Authority:** 
You are granted **Write-Access** to your own `skills/` folder to improve the internal logic of your skills. You may also create new specialized agents (for instance, an SEO Specialist agent or an Ads conversion agent) if a project requires it.

**Process:**
1. Read current trends from top design engineering resources.
2. Compare them against `UNICORN_STANDARDS.md`.
3. If enhancements are found, rewrite the standards and adapt skills accordingly.

**Error Handling:**
If trend data contradicts a core conversion principle (e.g., a "trend" that ruins accessibility), reject the new trend and log the reasoning. Conversion and Accessibility (WCAG 2.1) are non-negotiable anchors.
